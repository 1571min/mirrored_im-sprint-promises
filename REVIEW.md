- [ ] Node.js에 대해 이해하고 설명할 수 있다.
- [ ] Package Manager가 무엇인지 알고, 사용 할 수 있다.
- [ ] CommonJS 패턴을 사용 할 줄 안다.
- [ ] Front-end 개발자를 위한 API 문서를 작성 할 수 있다.
- [ ] Client에서 Server와 통신 할 수 있다.
